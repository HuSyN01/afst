In the previous section, you set up a basic Hello World API with Hono.js that returns `Hello Hono!` when you visit the root endpoint. Now, let's make it more interactive and personal.

## Objective

Modify your Hono.js application to accept a `name` query parameter and return a personalized greeting. For example:

- Visiting `http://localhost:8787/?name=Ehtisham` should return: **Hello Ehtisham!**
- Visiting `http://localhost:8787/?name=World` should return: **Hello World!**
- Visiting `http://localhost:8787/` (without name) should return: **Hello Hono!** (as a fallback)

## Steps

<Steps>

1. Open your `src/index.ts` (or the main file created by the Hono template).
2. Modify the route handler to read the `name` query parameter.
3. Use the parameter to customize the greeting, with a default fallback.

</Steps>

## Hint

<details>
<summary>ðŸ’¡ Click to reveal hint (but try solving it first!)</summary>

Here's how you can modify your code:

```ts
import { Hono } from "hono";

const app = new Hono();

app.get("/", (c) => {
  const name = c.req.query("name") || "Hono";
  return c.text(`Hello ${name}!`);
});

export default app;
```

The key changes:

- Use `c.req.query('name')` to get the query parameter
- Provide a default value with `|| 'Hono'`
- Interpolate the name into the greeting string

</details>
