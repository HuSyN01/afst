Before diving into the course, ensure you have these essential tools installed on your system. Each plays a crucial role in your development journey.

## Required Software

- **[Git](https://git-scm.com/install)**: Version control system for tracking changes and collaborating on code.
- **[Visual Studio Code (VS Code)](https://code.visualstudio.com)**: A powerful, free code editor with extensive extensions for web development.
- **[Bun](https://bun.sh/docs/installation)**: A fast JavaScript runtime and package manager used in our tech stack.
- **[Google Chrome](https://www.google.com/chrome)**: Modern web browser for testing and debugging your applications.
